# ゆーだいくん🐳 プロフィールサイト

BIGO LIVEで配信しているゆーだいくんのプロフィールサイトです。

公開URL: https://adayume1127.github.io/yudai-1192/

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | ページ全体の構成(ヒーロー・プロフィール・配信情報) |
| `style.css` | デザイン本体(配色・フォント・レイアウト) |
| `media.css` | スマホ・タブレット向けのレスポンシブ調整 |
| `normalize.css` | ブラウザ間の表示差異をリセットする標準CSS(変更不要) |
| `img/` | プロフィール画像 |

ai-sakura1582リポジトリをベースに作成しています。
